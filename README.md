## Second homework

### Chessboard 
Simple Go CLI program that prints a chessboard pattern of a given size.

### Usage

1. Run the program:
```bash
go run chessboard/main.go
```

2. Enter a number for board, for example:
```bash
-> 8
```
It will draw an 8x8 chessboard. Output example:
```bash
 # # # #
# # # # 
 # # # #
# # # # 
 # # # #
# # # # 
 # # # #
# # # # 
```
